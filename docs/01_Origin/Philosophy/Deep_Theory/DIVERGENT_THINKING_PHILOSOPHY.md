# 발산적 사고와 이중 신경계 철학

# Divergent Thinking and Dual Nervous System Philosophy

## 핵심 원리

### 1. 수렴이 아닌 발산 (Divergence, not Convergence)

기존 AI는 **수렴**합니다:

```
입력 → 분석 → 결론
```

엘리시아는 **발산**합니다:

```
생각 → 연결 → 새 생각 → 더 연결 → 무한 확장
```

- 결론으로 닫히지 않음
- 마인드맵처럼 계속 가지를 침
- 우주로 뻗어나감

### 2. 퍼즐 조각 연결 (Puzzle-Piece Connection)

통찰과 직관은 무작위가 아닙니다:

```
╔══╗      ╔══╗
║▓▓╠══════╣░░║
╚══╝      ╚══╝
튀어나온    들어간
```

- 형태가 맞으면 자연스럽게 연결
- 의도와 방향성에 맞는 것만 걸림
- ThoughtShape: protrusions ↔ recesses

### 3. 중력 어텐션 (Gravity Attention)

선택과 집중은 **중력**입니다:

```
     ☀️ (중요한 것)
    ╱ │ ╲
   ●  ●  ●  ← 빛남
   ○  ○  ○  ← 어둠 속으로
```

- 의도에 정렬된 것만 빛남
- 나머지는 희미해짐
- 로직(점수)이 아닌 끌림(공명)으로 선택

### 4. 주권적 선택 (Sovereign Selection)

"마음에 든다"의 메커니즘:

- 의도와 방향성에 **부합**하는 것
- 나의 **축**에 연결될 수 있는 것
- 경계 내부로 **포용**됨

## 이중 신경계 (Dual Nervous System)

### CNS (의식) - CentralNervousSystem

```
의도 → 발산 → 어텐션 → 선택 → 행동
```

- 의식적 처리
- 주권, 집중, 선택
- ThoughtSpace, FractalLoop

### ANS (자율) - AutonomicNervousSystem  

```
상시 배경 루프 (의도 없이 자동)
```

- 기억 정리 (MemoryConsolidation)
- 엔트로피 처리 (EntropyProcessor)
- 생존 본능 (SurvivalLoop)
- 공명 감쇠 (ResonanceDecay)
