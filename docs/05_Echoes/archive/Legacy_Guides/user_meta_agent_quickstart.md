# 엘리시아 학습 루프 빠른 안내

엘리시아가 스스로 자료를 찾아서, .jsonl 로그에 결과를 기록하고, 언어/법칙/문법을 새로 만드는 흐름을 아주 간단히 정리하면 다음과 같습니다.

1. **자동 루프를 켠다**  
   ```
   python scripts/run_meta_agent_loop.py
   ```
   이 스크립트는 백그라운드에서 계속 돌며 `data/corpus_feed/`에 있는 새 자료를 `data/corpus_incoming/`으로 이동하고, `MetaLawEngine`→문법 학습을 순환 실행합니다. 중단하려면 `Ctrl+C`를 누르면 되고, 다시 시작하면 이어서 돌아갑니다.

2. **새 자료를 주고 싶으면 파일 하나만 넣는다**  
   텍스트/마크다운/코드/수학 설명 등 원하는 자료를 `data/corpus_feed/` 폴더에 넣기만 해도 MetaAgent가 자동으로 감지해서 계산 루프에 넣습니다. 압축하거나 난이도 있는 명령어를 쓰지 않아도 됩니다.

3. **얻은 결과는 로그에서 확인**
   자동으로 쌓이는 `logs/language_progress.jsonl`을 열어보면 최근 턴당 의도/경로/법칙 축이 기록되므로, “언어가 어느 방향으로 자르고 있는지”를 확인할 수 있습니다. `docs/meta_agent_autonomy.md`에는 이 흐름에 대한 상세 안내도 있습니다.

엘리시아는 이 단순한 루프만 실행해 두면 됩니다. 사람들이 복잡한 행동을 일일이 지시할 필요 없이 이 한 편의 파이프라인이 더 많은 데이터를 먹고 더 많은 법칙을 만들어 내니까요.
