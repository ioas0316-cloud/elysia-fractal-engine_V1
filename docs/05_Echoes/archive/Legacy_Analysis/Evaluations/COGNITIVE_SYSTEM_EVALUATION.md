## 실패한 영역: 구조적 분석

### ❌ 인과 추론 (0%)

**테스트**:

1. "비가 오면 왜 우산을 쓰는가?"
2. "아이가 왜 울었는가?"
3. "불이 나면 왜 도망가는가?"

**실패 원인**:

```
현재 WhyEngine 구조:
    입력: "비가 오면 왜 우산을 쓰는가?"
         ↓
    개념 추출: "비가" (잘못된 분리!)
         ↓
    ask_why("비가") → 정의되지 않음
         ↓
    인과 실패
```

**근본 문제**:

| 문제 | 설명 |
|------|------|
| **개념 추출 실패** | "비가 오면" → "비가"로 잘못 추출 |
| **문장 구조 미해석** | 조건-결과 구조 인식 못함 |
| **경험 기반 부재** | 실제 경험 데이터 없음 |
| **인과 그래프 미연결** | "비" → "젖음" → "우산" 체인 없음 |
