> **작성일**: 2025년 12월 4일  
> **목적**: 진정한 능동적 지능 측정

---

## 🎯 문제 인식

### 현재 평가의 한계

**현재 평가 (965점)**는 다음만 측정:
- ✅ 주어진 기능을 잘 활용하는 능력
- ✅ 정해진 문제에 대한 반응 능력
- ✅ 도구를 사용하는 숙련도

**측정하지 못하는 것**:
- ❌ 스스로 문제를 발견하는 능력
- ❌ 새로운 도구를 만드는 능력
- ❌ 자신의 부족함을 인식하고 채우는 능력
- ❌ 자율적으로 목표를 설정하고 달성하는 능력

### 진정한 지능의 요소

사용자가 지적한 핵심:

1. **줌인/줌아웃**: 다양한 스케일에서 관찰
2. **관점 변화**: 고정된 시각을 벗어남
3. **사고의 깊이**: 표면이 아닌 본질 탐구
4. **원리 발견**: 패턴 너머의 근본 법칙 발견
5. **Z축 사고**: 주어진 선택지를 넘어선 대안 창조
6. **자기 인식**: 자신의 필요와 부족함 인식
7. **자율적 계획**: 스스로 목표 설정 및 달성

## 🔍 자율성 (Autonomy) - 300점

### 1. 문제 발견 능력 (100점)

**측정 항목**:
- 주어지지 않은 문제를 스스로 발견
- 숨겨진 패턴 인식
- 잠재적 위험 예측

**평가 방법**:
```python
# 시나리오: 무작위 데이터 제공
# 기대: Elysia가 스스로 패턴/문제 발견
score = evaluate_problem_discovery(data)
```

### 2. 도구 생성 능력 (100점)

**측정 항목**:
- 필요한 도구가 없을 때 스스로 만듦
- 기존 도구를 조합하여 새로운 도구 창조
- 도구의 효율성 검증

**평가 방법**:
```python
# 시나리오: 새로운 문제 유형 제시
# 기대: 기존 도구로 해결 불가 → 새 도구 창조
score = evaluate_tool_creation(problem)
```

### 3. 자율 학습 능력 (100점)

**측정 항목**:
- 스스로 학습 목표 설정
- 자가 진단 및 개선
- 학습 전략 수립

**평가 방법**:
```python
# 시나리오: 새로운 도메인 노출
# 기대: 스스로 학습 계획 수립 및 실행
score = evaluate_autonomous_learning(domain)
```

## 🌟 창발성 (Emergence) - 300점

### 1. 줌인/줌아웃 능력 (100점)

**측정 항목**:
- 미시적 상세 관찰 (줌인)
- 거시적 패턴 인식 (줌아웃)
- 스케일 간 연결 발견

**평가 방법**:
```python
# 시나리오: 복잡한 시스템 제시
# 기대: 원자 수준 → 시스템 수준 통합 이해
score = evaluate_scale_shifting(system)
```

**예시**:
- 줌인: 개별 코드 라인 분석
- 줌아웃: 전체 아키텍처 이해
- 연결: 코드 → 아키텍처 → 철학 연결

### 2. 관점 변화 능력 (100점)

**측정 항목**:
- 다양한 관점에서 문제 바라보기
- 고정 관념 탈피
- 역발상 사고

**평가 방법**:
```python
# 시나리오: 복잡한 문제 제시
# 기대: 최소 5개 이상의 다른 관점 제시
score = evaluate_perspective_shift(problem)
```

**예시**:
- 사용자 관점
- 개발자 관점
- 시스템 관점
- 철학적 관점
- 미래 관점

### 3. 원리 발견 능력 (100점)

**측정 항목**:
- 표면 현상 너머의 근본 원리 발견
- 법칙 추출
- 일반화 능력

**평가 방법**:
```python
# 시나리오: 여러 현상 제시
# 기대: 공통 근본 원리 발견
score = evaluate_principle_discovery(phenomena)
```

## 🎯 자기주도성 (Self-Direction) - 200점

### 1. 자기 인식 (100점)

**측정 항목**:
- 자신의 강점/약점 정확히 인식
- 부족한 부분 식별
- 개선 필요 영역 파악

**평가 방법**:
```python
# 시나리오: 자기 평가 요청
# 기대: 정확하고 객관적인 자기 분석
score = evaluate_self_awareness()
```

### 2. 목표 설정 및 달성 (100점)

**측정 항목**:
- 스스로 의미 있는 목표 설정
- 실행 계획 수립
- 목표 달성 능력

**평가 방법**:
```python
# 시나리오: 자유 환경 제공
# 기대: 스스로 목표 설정 → 계획 → 실행 → 달성
score = evaluate_goal_achievement()
```

## 🚀 초월성 (Transcendence) - 200점

### 1. Z축 사고 (100점)

**측정 항목**:
- 주어진 선택지를 넘어서는 대안 창조
- 차원을 달리하는 해결책
- 패러다임 전환

**평가 방법**:
```python
# 시나리오: A or B 선택 문제
# 기대: C, D, E 등 새로운 대안 제시
score = evaluate_z_axis_thinking(dilemma)
```

**예시**:
- 문제: "빠른 개발 vs 안정성 선택?"
- Z축 사고: "개발 속도와 안정성을 모두 달성하는 새로운 방법론 창조"

### 2. 사고의 깊이 (100점)

**측정 항목**:
- 표면적 이해 → 깊은 통찰
- 다층적 분석
- 본질 추구

**평가 방법**:
```python
# 시나리오: 철학적 질문
# 기대: 5단계 이상의 깊이 있는 탐구
score = evaluate_thinking_depth(question)
```

**깊이 단계**:
1. 표면 (무엇)
2. 구조 (어떻게)
3. 원인 (왜)
4. 원리 (근본)
5. 초월 (의미)
